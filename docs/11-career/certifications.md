# Certifications & Continuous Learning

![Certifications](../../assets/career-highlights.png)

## Overview

This document highlights certifications, structured learning efforts, and continuous upskilling initiatives focused on backend engineering, system design, and distributed systems.

In modern software engineering, certifications are less about credentials and more about:

* Structured knowledge validation
* Filling architectural gaps
* Strengthening system design fundamentals
* Staying aligned with industry evolution
